# wicket-gate
Контроллер турникета для оборудования никирет (я назвал ее СКД Прометей)

* client предназначен для запуска на MCU Tion Pro 28 производства ЗАО ЗЕО
* service для x86-64  архитектуры
* widget x86-64 утилита для визуального контроля и управления
* crossboard nikiret самостоятельная реализация кроссплаты никирет на MCU PIC24FJ64GA104 с протокол никирет
